# TaskFlow Core (OSS)

Open-source task management application built with NestJS, Next.js, Expo, and Electron.

## Getting Started

```bash
cd core
bun install
bun run dev
```

## Apps

- **api** — NestJS backend (port 4000)
- **web** — Next.js frontend (port 3000)
- **mobile** — Expo (React Native)
- **desktop** — Electron wrapper

## Packages

- **@todoist/shared** — Shared types and utilities

## Docker

```bash
docker-compose up
```
