# Features

## Core Features (OSS)

### Task Management
- Create, update, delete, and complete tasks
- **Priority levels**: low, medium, high, urgent
- **Due dates** with date filtering
- **Labels** for flexible categorization
- **Subtasks** — nested task hierarchy
- **Recurrence** — daily, weekly, monthly, yearly with configurable intervals

### Projects
- Organize tasks into projects with color and icon customization
- Project statuses and categories
- Tags for cross-project organization
- Start date and target launch date tracking

### Comments & Collaboration
- Comment on tasks
- Activity history tracking

### Goals
- Set and track goals tied to projects and tasks

### Connections
- External service integrations

### Multi-Platform
- **Web** — Next.js 15 with responsive design
- **Mobile** — React Native via Expo for iOS and Android
- **Desktop** — Tauri for native macOS, Windows, and Linux apps
- **API** — RESTful NestJS backend with Swagger docs

### Self-Hosting
- Docker Compose deployment
- MongoDB for data storage
- Clerk for authentication

## Cloud Features (SaaS)

- **AI Agents** — LLM-powered task decomposition and execution
- **Team Collaboration** — Shared projects and real-time sync
- **Advanced Integrations** — Calendar sync, Slack, GitHub
- **Managed Infrastructure** — No deployment needed

See [Cloud Overview](/cloud/overview) for details.
