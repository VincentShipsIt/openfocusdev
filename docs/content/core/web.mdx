# Web App

The TaskFlow web app is built with **Next.js 15** and **React 19**.

## Running

```bash
cd core
bun run dev:web
```

Opens on `http://localhost:3000`.

## Stack

- **Next.js 15** — App Router, Server Components
- **React 19** — Latest React with concurrent features
- **Clerk** — Authentication UI components and session management
- **REST Client** — Communicates with the NestJS API

## Environment Variables

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_xxxxx
NEXT_PUBLIC_API_URL=http://localhost:4000
```

## Building for Production

```bash
cd core/apps/web
bun run build
bun run start
```

## Docker

The web app has its own Dockerfile at `apps/web/Dockerfile`. When using `docker compose up`, it runs on port 3000 and connects to the API container automatically.
