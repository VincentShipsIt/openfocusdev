# Desktop App

The TaskFlow desktop app is built with **Tauri**, wrapping the React frontend in a native window.

## Running

```bash
cd core
bun run dev:desktop
```

## Prerequisites

- [Tauri prerequisites](https://tauri.app/start/prerequisites/) for your OS
- Rust toolchain

## Supported Platforms

- macOS (Apple Silicon + Intel)
- Windows
- Linux

## Architecture

Tauri provides a lightweight native shell (~5MB) that renders the web frontend. The desktop app connects to the same API as the web and mobile apps.

## Building

```bash
cd core/apps/desktop
bunx tauri build
```

This produces platform-specific installers (`.dmg`, `.msi`, `.AppImage`).
