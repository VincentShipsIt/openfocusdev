# TaskFlow

**Open-source task management for developers.** A full-stack Todoist alternative built with NestJS, Next.js, React Native, and Tauri — with optional AI-powered task agents.

## What is TaskFlow?

TaskFlow is a monorepo task management platform with two tiers:

- **TaskFlow Core (OSS)** — Self-hostable API, web app, mobile app (Expo), and desktop app (Tauri). MIT licensed.
- **TaskFlow Cloud** — Managed SaaS with AI agents, team collaboration, and integrations built on top of Core.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| API | NestJS + MongoDB |
| Web | Next.js 15 + React 19 |
| Mobile | React Native (Expo) |
| Desktop | Tauri + React |
| Auth | Clerk |
| AI | LLM-powered task agents |
