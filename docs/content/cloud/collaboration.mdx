# Collaboration

TaskFlow Cloud enables team-based task management with shared workspaces.

## Team Workspaces

- Create workspaces for your team
- Invite members via email
- Role-based access control (owner, admin, member)

## Shared Projects

Projects in Cloud can be shared across team members:

- Assign tasks to team members
- Real-time updates — changes sync instantly
- Comment threads on tasks for async discussion

## Activity Feed

Track what's happening across your team:

- Task creation, completion, and reassignment
- Comment activity
- Project milestones

## Integrations

- **Slack** — Get notifications in your Slack channels
- **GitHub** — Link tasks to PRs and issues
- **Calendar** — Sync due dates with Google Calendar or Outlook
