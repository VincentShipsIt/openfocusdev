# AI Agents

TaskFlow Cloud includes an AI agent system that can decompose, plan, and execute tasks automatically.

## How It Works

When you create a task with `aiEnabled: true` and provide an `aiPrompt`, the AI agent system:

1. **Analyzes** the task prompt and context
2. **Decomposes** complex tasks into subtasks
3. **Executes** each subtask, updating progress in real-time
4. **Reports** results back to the task

## Creating an AI Task

```bash
curl -X POST http://api.taskflow.dev/tasks \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Research competitor pricing",
    "aiEnabled": true,
    "aiPrompt": "Research the top 5 task management apps and create a comparison table of their pricing tiers",
    "projectId": "project_123"
  }'
```

## Triggering AI Execution

Once created, trigger the AI agent:

```bash
curl -X POST http://api.taskflow.dev/tasks/:id/ai-execute \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Additional context or instructions"
  }'
```

## Task Schema Fields

| Field | Type | Description |
|-------|------|-------------|
| `aiEnabled` | `boolean` | Enable AI agent for this task |
| `aiPrompt` | `string` | Instructions for the AI agent |
| `aiStatus` | `string` | Current AI execution status |
| `aiResult` | `object` | AI execution output |

## Limitations

- AI agents are a Cloud-only feature
- Complex tasks may take longer to process
- Results should be reviewed by a human before marking complete
