# TaskFlow Cloud

TaskFlow Cloud is the managed SaaS layer built on top of TaskFlow Core. It adds AI-powered features, team collaboration, and hosted infrastructure so you don't have to manage your own deployment.

## What Cloud Adds

| Feature | Core (OSS) | Cloud |
|---------|-----------|-------|
| Task management | ✅ | ✅ |
| Projects, goals, comments | ✅ | ✅ |
| Web, mobile, desktop apps | ✅ | ✅ |
| Self-hosting | ✅ | Managed |
| AI task agents | ❌ | ✅ |
| Team workspaces | ❌ | ✅ |
| Real-time collaboration | ❌ | ✅ |
| Calendar & Slack integrations | ❌ | ✅ |
| Priority support | ❌ | ✅ |

## Architecture

Cloud runs the same Core API and web app with additional modules:

- **AI Agent Service** — processes tasks with `aiEnabled: true` and executes them via LLM chains
- **Collaboration Service** — real-time sync, shared projects, team permissions
- **Integration Service** — third-party connections (calendar, Slack, GitHub)
