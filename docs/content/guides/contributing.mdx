# Contributing

TaskFlow Core is open-source and welcomes contributions.

## Setup

```bash
git clone https://github.com/shipshitdev/todo.git
cd todo/core
bun install
```

## Development Workflow

1. Create a feature branch: `git checkout -b feat/my-feature`
2. Make your changes
3. Run checks: `bun run check`
4. Run tests: `bun run test`
5. Format code: `bun run format`
6. Commit and push
7. Open a pull request

## Code Style

TaskFlow uses [Biome](https://biomejs.dev) for linting and formatting:

```bash
bun run lint      # Check for issues
bun run format    # Auto-format code
bun run check     # Both lint + format check
```

Husky + lint-staged run Biome automatically on commit.

## Testing

Tests use **Vitest** with Testing Library for React components:

```bash
bun run test              # Run all tests
bun run test:coverage     # With coverage
```

Test files are colocated with source code in `__tests__/` directories.
