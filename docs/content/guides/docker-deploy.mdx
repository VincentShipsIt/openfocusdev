# Docker Deployment

Deploy TaskFlow with Docker Compose for production use.

## Prerequisites

- Docker and Docker Compose installed
- Clerk API keys

## docker-compose.yml

The `core/docker-compose.yml` defines three services:

```yaml
services:
  mongodb:
    image: mongo:7
    ports:
      - '27017:27017'
    volumes:
      - mongodb_data:/data/db
    healthcheck:
      test: echo 'db.runCommand("ping").ok' | mongosh --quiet
      interval: 10s
      timeout: 5s
      retries: 5

  api:
    build:
      context: .
      dockerfile: apps/api/Dockerfile
    ports:
      - '4000:4000'
    environment:
      - MONGODB_URI=mongodb://mongodb:27017/taskflow
      - PORT=4000
      - FRONTEND_URL=http://localhost:3000
      - CLERK_SECRET_KEY=${CLERK_SECRET_KEY}
    depends_on:
      mongodb:
        condition: service_healthy

  web:
    build:
      context: .
      dockerfile: apps/web/Dockerfile
    ports:
      - '3000:3000'
    environment:
      - NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=${NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY}
      - NEXT_PUBLIC_API_URL=http://api:4000
    depends_on:
      - api

volumes:
  mongodb_data:
```

## Running

```bash
cd core

# Set your Clerk keys
export CLERK_SECRET_KEY=sk_live_xxxxx
export NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_live_xxxxx

# Start all services
docker compose up -d

# Check status
docker compose ps

# View logs
docker compose logs -f api
```

## Production Considerations

### Reverse Proxy

Put nginx or Caddy in front for HTTPS:

```nginx
server {
    listen 443 ssl;
    server_name taskflow.yourdomain.com;

    location / {
        proxy_pass http://localhost:3000;
    }

    location /api/ {
        proxy_pass http://localhost:4000/;
    }
}
```

### MongoDB Security

For production, add authentication to MongoDB:

```yaml
mongodb:
  image: mongo:7
  environment:
    MONGO_INITDB_ROOT_USERNAME: admin
    MONGO_INITDB_ROOT_PASSWORD: ${MONGO_PASSWORD}
```

Update `MONGODB_URI` accordingly:
```
MONGODB_URI=mongodb://admin:password@mongodb:27017/taskflow?authSource=admin
```

### Backups

Schedule MongoDB backups:

```bash
docker compose exec mongodb mongodump --out /backup/$(date +%Y%m%d)
```

## Updating

```bash
git pull

docker compose up -d
```
