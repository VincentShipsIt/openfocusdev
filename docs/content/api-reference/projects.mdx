# Projects API

All endpoints require authentication via a Clerk Bearer token.

**Base URL:** `http://localhost:4000`

## Create Project

```http
POST /projects
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | `string` | Yes | Project name |
| `description` | `string` | No | Project description |
| `color` | `string` | No | Hex color code |
| `icon` | `string` | No | Icon identifier |
| `status` | `string` | No | Project status |
| `category` | `string` | No | Project category |
| `startDate` | `string` | No | ISO 8601 start date |
| `targetLaunchDate` | `string` | No | ISO 8601 target launch date |
| `tags` | `string[]` | No | Project tags |

### Example

```bash
curl -X POST http://localhost:4000/projects \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Website Redesign",
    "description": "Complete overhaul of the marketing site",
    "color": "#4F46E5",
    "category": "engineering",
    "targetLaunchDate": "2025-09-01T00:00:00.000Z",
    "tags": ["marketing", "design"]
  }'
```

## List Projects

```http
GET /projects
```

### Query Parameters

| Param | Type | Description |
|-------|------|-------------|
| `page` | `number` | Pagination page |
| `limit` | `number` | Items per page |

```bash
curl http://localhost:4000/projects \
  -H "Authorization: Bearer $TOKEN"
```

## Get Project

```http
GET /projects/:id
```

## Update Project

```http
PATCH /projects/:id
```

Accepts the same fields as Create (all optional).

## Delete Project

```http
DELETE /projects/:id
```
